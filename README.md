## 平台简介

IoT Gateway Platform 是一款功能强大的智能物联网网关管理平台，致力于为企业提供统一的设备接入、数据采集、协议转换和设备管理解决方案。

### 核心优势

- **多协议支持** - 无缝对接 Modbus TCP/RTU、S7、BACnet 等主流工业协议，实现异构设备统一接入
- **可视化配置** - 直观的规则编排界面，无需编程即可完成设备配置和自动化规则
- **实时监控** - 设备状态实时展示，数据采集日志全程可追溯，故障快速定位
- **灵活扩展** - 插件化架构设计，支持自定义协议和业务逻辑扩展
- **安全可靠** - 集成国密加密算法，符合等保要求，保障数据传输安全

### 适用场景

- **智能制造** - 生产设备数据采集与监控，生产线自动化控制
- **智慧园区** - 能源管理与环境监测，智能设备统一接入
- **智慧农业** - 农业传感器数据汇聚，环境参数自动调控
- **智能楼宇** - 楼宇自控系统集成，BA系统协议转换

## 快速启动

### 环境准备

| 组件 | 版本要求 | 用途 |
|------|----------|------|
| Node.js | ≥ 18 | 前端运行环境 |
| JDK | 17 | 后端运行环境 |
| Maven | 最新版 | 依赖管理工具 |
| Redis | 最新版 | 缓存服务 |
| MySQL | 8.0 / 5.7 | 数据存储 |

### 前端启动

进入前端目录：
```bash
cd snowy-admin-web
```

安装依赖：
```bash
npm install
```

启动开发服务：
```bash
npm run dev
```

### 后端启动

1. 配置数据库连接信息
2. 配置 Redis 连接信息
3. 在开发工具中启动主模块 `snowy-web-app`

## 项目结构

本项目采用插件化架构，实现模块分离和高度解耦：

```
iot_gateway
  ├─ snowy-admin-web          # 前端管理平台
  │   ├─ src
  │   │   ├─ api               # API 接口定义
  │   │   ├─ components        # 公共组件
  │   │   ├─ views             # 页面视图
  │   │   ├─ router            # 路由配置
  │   │   └─ utils             # 工具类
  ├─ snowy-common             # 基础公共模块
  ├─ snowy-plugin             # 功能插件模块
  │   ├─ snowy-plugin-auth    # 认证鉴权
  │   ├─ snowy-plugin-biz     # 业务功能
  │   ├─ snowy-plugin-dev     # 开发工具
  │   ├─ snowy-plugin-iot     # IoT 设备管理
  │   ├─ snowy-plugin-sys     # 系统管理
  │   └─ snowy-plugin-mobile  # 移动端管理
  ├─ snowy-plugin-api         # 插件 API 接口
  └─ snowy-web-app            # 主启动模块
```

## 功能特性

### 设备管理
- **多协议支持** - Modbus TCP/RTU、S7、BACnet 等主流工业协议
- **设备全生命周期** - 设备注册、配置、监控、报警、维护
- **实时状态** - 设备在线/离线状态实时监控
- **批量操作** - 设备导入导出、批量配置

### 产品管理
- **物模型定义** - 灵活定义设备属性、命令、报警
- **寄存器映射** - 可视化配置协议寄存器与物模型的映射关系
- **设备模板** - 支持产品级配置模板，快速复制应用

### 规则引擎
- **可视化编排** - 拖拽式流程设计，无需编码
- **完整性检查** - 自动校验规则配置完整性，防止配置错误
- **设备触发** - 设备数据变化触发自动化流程
- **条件判断** - 支持简单条件和复杂条件组，自动继承设备上下文
- **动作执行** - 支持设备控制、数据上报等动作
- **日志追踪** - 完整的规则执行日志，问题快速定位

### 数据管理
- **实时数据** - 设备数据实时采集与展示
- **历史数据** - 数据存储与查询，支持时间范围筛选
- **数据解析** - 协议数据自动解析与类型转换
- **数据上报** - 支持向云平台上报设备数据

### 运维管理
- **系统监控** - 实时监控网关运行状态
- **日志管理** - 采集日志、操作日志全程记录
- **权限控制** - 基于 RBAC 的细粒度权限管理
- **配置备份** - 支持配置导入导出，快速迁移恢复

## 技术架构

### 后端技术栈
- Spring Boot 3.x - 应用框架
- MyBatis Plus - ORM 框架
- Redis - 缓存中间件
- MySQL - 关系型数据库
- SM2/SM3/SM4 - 国密加密算法

### 前端技术栈
- Vue 3 - 渐进式框架
- Ant Design Vue 4 - UI 组件库
- Vite 5 - 构建工具
- Pinia - 状态管理
- Axios - HTTP 客户端
